# Ellipse Bounce

A physics simulation of a ball bouncing inside an ellipse.

## Files

- `ellipse_bounce.py` - Interactive animation showing a ball bouncing inside an elliptical boundary
- `ellipse_bounce_record.py` - Script to record the animation as a GIF
- `ellipse_bounce.gif` - Pre-recorded animation

## Run

```bash
python ellipse_bounce.py
```

## Requirements

- numpy
- matplotlib
